# Contributing to File Organiser

First off, thanks for taking the time to contribute! 👋  
This project is open source because we want it to get better with help from the community.  

---

## 🌱 How to Contribute

- **Found a bug?**  
  Open an issue describing it clearly (what happened, what you expected, steps to reproduce if possible).  

- **Want to add a feature?**  
  Open an issue first to discuss it. Not every idea will fit, and that’s okay.  

- **Ready to code?**  
  1. Fork the repo.  
  2. Make your changes on a new branch.  
  3. Submit a pull request (PR) with a clear description of what you did and why.  

---

## 💡 Guidelines

✅ Be kind and respectful.  
✅ Keep pull requests focused – one thing at a time.  
✅ Follow existing code style where possible.  
✅ Test your changes if you can.  

---

## 🚫 Please Don’t:

- Send low-effort or spammy requests (“add my name to contributors” etc).  
- Start working on big features without discussing them first.  
- Submit code you didn’t write yourself (unless clearly marked and licensed).
